# MicroFrontendTutorial

Application developed using React, vue and svelte programming languages and Webpack 5's new Module Federation

## Installation

Install the dependencies and devDependencies and start the server.

```sh
cd headers
npm install
npm start
```

```sh
cd body
npm install
npm start
```

```sh
cd products
npm install
npm start
```

```sh
cd footer
npm install
npm start
```
